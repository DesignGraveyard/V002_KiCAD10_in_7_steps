# V002_KiCAD10_in_7_steps
Master KiCad 10 in just 7 steps! In this tutorial, we design a custom, ultra-compact ESP32-S3 USB development board from scratch. From setting up your schematic to routing a 2-layer board with a custom PCB edge-connector, this video covers everything you need to know to get started with the latest features in KiCad 10.
